# ProjectScope Preview（项目成效预览器）

一个轻量级项目预览 demo，用来快速体验“填写项目信息 -> 生成基础方案 -> 选择下一步”的流程。

## 使用方式

下载仓库后，直接用浏览器打开：

```text
index.html
```

## 你可以体验

- 选择项目类型和当前阶段
- 填写目标用户和主要目标
- 生成一份基础开发方向
- 在下一步中选择自主推进或申请项目诊断

## 适用场景

- 想先判断一个项目是否值得继续做
- 想快速整理一个 MVP 的基础方向
- 想把想法转成一个可讨论的项目雏形

## 说明

本仓库仅提供公开 demo，用于基础流程预览。正式项目需要结合真实需求、现有材料、预算和交付周期进一步评估。
